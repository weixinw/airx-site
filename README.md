# AIRX Website

The official website for AIRX, the second phase of BRC (Brain Robotics Capital). AIRX is a VC + SPV + Incubation platform that partners with scientists and tech entrepreneurs to build legendary companies.

## About AIRX

AIRX goes beyond traditional passive LP investment. We function as fellow co-founders, providing:

- **Expert Network**: Domain expertise in engineering, manufacturing, and supply chain
- **Go-to-Market Support**: Operational and execution expertise to validate business models
- **Financial Connections**: Access to investment banks for IPO, M&A, and exit opportunities

## Development

This is a Next.js application optimized for static export and GitHub Pages deployment.

### Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
```

This generates a static export in the `out` directory.

## Deployment

The site is automatically deployed to GitHub Pages via GitHub Actions when changes are pushed to the `main` branch.

## Technology Stack

- **Next.js 14** - React framework with static export
- **TypeScript** - Type safety
- **CSS Modules** - Scoped styling
- **GitHub Pages** - Hosting

## Portfolio

The website showcases the track record from BRC Phase I portfolio companies, including successful exits and acquisitions.
